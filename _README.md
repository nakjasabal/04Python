# 마크다운 표기법 

| 기호 | 설명 | 표시 예 |
|---|---|---|
| `#` | 제목 | `# 제목 1` |
| `**` | 굵은 글씨 | `**굵은 글씨가 됩니다**` |
| `*` | 기울임꼴 | `*기울임꼴이 됩니다*` |
| `***` | 굵은 글씨 및 기울임꼴 | `***굵은 글씨 및 기울임꼴***` |
| `>` | 인용 | `> 인용문입니다.` |
| `1.` | 번호 매기기 목록 | `1. 사과` |
| `-` | 목록 | `- 사과` |
| `` ` `` | 인라인 코드 | `` `console.log("Hello");` `` |
| ` ``` ` | 코드 블록 | 테이블 내 표현 생략 |
| `---` | 수평선 | `---` |
| `[ ]( )` | 링크 | `[Google](https://google.com)` |
| `![ ]( )` | 이미지 | `![이미지](image.png)` |
| `\` | 이스케이프 문자 | `\*` 또는 `\\` |
| `(생략)` | 테이블 | 테이블 형식으로 표시 |
| `- [ ]` | 할 일 목록 | `- [ ] 미완료 할 일` |
| `- [x]` | 완료된 할 일 | `- [x] 완료된 할 일` |
| `==` | 하이라이트 | `==하이라이트==` |
| `~` | 아래 첨자 | `H~2~O` |
| `^` | 위 첨자 | `E = mc^2^` |

# 제목 만들기

제목은 H2 아래에 H3이, H3 아래에 H4가 오는 것과 같은 계층 구조를 나타내는 요소입니다. 마크다운에서는 줄의 시작 부분에 # 기호를 붙여 제목을 만듭니다. #의 수가 많을수록 작은 제목이 됩니다.

# 이것은 가장 큰 제목 (레벨 1)

## 이것은 2번째 제목 (레벨 2)

### 이것은 3번째 제목 (레벨 3)

#### 이것은 4번째 제목 (레벨 4)

##### 이것은 5번째 제목 (레벨 5)

###### 이것은 가장 작은 제목 (레벨 6)

---

# 단락
단락은 문서의 기본 단위입니다. 마크다운에서는 빈 줄로 단락을 구분합니다.


이것은 새로운 단락입니다.
빈 줄에 의해 단락이 나뉘어 있습니다.

---

# 굵은 글씨 (Bold)
굵은 글씨로 만들고 싶다면, 글자를 두 개의 별표 **로 감쌉니다.
이 문장에는 **중요한 부분**이 있습니다.

---

# 기울임꼴 (Italic)
기울임꼴로 만들고 싶다면, 글자를 하나의 별표 *로 감쌉니다.
이 문장에는 *강조하고 싶은 부분*이 있습니다.

---

# 굵은 글씨 및 기울임꼴 (Bold and Italic)
굵은 글씨 및 기울임꼴로 만들고 싶다면, 글자를 세 개의 별표 ***로 감쌉니다.
이 문장에는 ***매우 중요한 부분***이 있습니다.

---

# 취소선
실수했거나 삭제된 내용을 나타내기 위해 사용합니다. 두 개의 물결표(~~)로 글자를 감쌉니다.

~~이 정보는 오래되었습니다.~~ 최신 정보는 여기입니다.

---

# 하이라이트
중요한 글자를 형광펜으로 표시한 것처럼 보여줍니다. 두 개의 등호 ==로 감쌉니다.

이 ==매우 중요한 부분==을 하이라이트해야 합니다.

이 <mark>매우 중요한 부분</mark>을 하이라이트합니다.

---

# 인용문
다른 사람의 말이나 중요한 문장을 인용할 때는 인용 기호를 사용합니다. 줄의 시작 부분에 >를 붙입니다. 이것은 인용문입니다. 다른 사람의 말이나 중요한 문장을 이렇게 표시할 수 있습니다.

> 이것은 *인용문*입니다.
> 다른 사람의 말이나 **중요한** 문장을
> 이렇게 표시할 수 있습니다.
>> 이것은 인용문 안의 인용문입니다.

---

# 목록 만들기
정보를 정리하여 표시하기 위해 리스트를 사용합니다. 번호 매기기 목록은 각 항목 앞에 숫자와 점을 붙이고, 점 뒤에 공백을 넣으면 목록이 됩니다.

1. 첫번째 항목
2. 두번째 항목
1. 세번째 항목
1. 네번째 항목 

순서 없는 목록은 항목 앞에 -, *, + 중 하나를 붙입니다. 이 중 하나로 통일하는 것이 좋습니다.
- 항목A
- 항목B
- 항목C

리스트는 중첩하는 것이 가능합니다. 하위 항목은 4개의 공백 또는 1개의 탭으로 들여쓰기 합니다.

1. 주요 항목 1
   - 부 항목 1-1
   - 부 항목 1-2
2. 주요 항목 2
   - 부 항목 2-1
   - 더욱 세부적인 항목

# 코드 블록
코드 블록은 백틱 3개( ``` )로 감쌉니다.

```python
def greetings():
	print("Hello, World!")
```

# 링크

기본적인 링크는 링크 텍스트를 [ ]로 묶고, URL을 ( )로 묶습니다. Windows의 경우 Ctrl+K, macOS의 경우 Cmd+K로 링크 서식을 지정할 수 있습니다.

검색 엔진은 [Google](https://google.com) 입니다. 

제목이 있는 링크는 URL 뒤에 제목을 추가할 수 있습니다. 마우스 커서를 링크 위에 올리면 표시됩니다.

검색 엔진은 [Google](https://google.com "Google입니다") 입니다.

자동 링크는 URL이나 이메일 주소를 <>로 감싸면 자동으로 링크가 됩니다.

<https://google.com/>

<example@email.com>

# 테이블

데이터를 정리하여 표시하기 위해 테이블을 사용합니다. 파이프(|)로 열을 구분하고, 하이픈(-)으로 헤더 행을 만듭니다.

테이블 내에서는 링크나 강조 등의 마크다운 표기법도 사용할 수 있습니다. 단, 제목이나 목록, 이미지는 사용할 수 없습니다.

| 이름 | 나이 | 직업 |
|------|------|------| 
| 홍길동 | 20 | 엔지니어 |
| 이몽룡 | 25 | 디자이너 |

왼쪽 정렬, 중앙 정렬, 오른쪽 정렬

:--- 왼쪽 정렬
:---: 중앙 정렬
---: 오른쪽 정렬

| 왼쪽 정렬 하기 | 중앙 정렬 하기 | 오른쪽 정렬 하기 |
|:----------|:---------:|----------:|
| 왼쪽      | 중앙      | 오른쪽      |
| 데이터    | 데이터    | 데이터      |
